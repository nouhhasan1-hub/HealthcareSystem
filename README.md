# Healthcare Management System

University of Hertfordshire - Software Architecture Assignment

## Project Overview
A healthcare management system implementing MVC pattern and Singleton design pattern for UK primary and secondary care integration.

## Technology Stack
- Java 17+
- Java Swing (GUI)
- CSV file storage
- Git for version control

## Development Timeline
- Week 1: Entity classes and data loading
- Week 2: Business logic and services
- Week 3: GUI implementation
- Week 4: Integration and testing
